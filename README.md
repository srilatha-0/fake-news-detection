# 📰 Fake News Detection using Machine Learning

This project detects whether a news article is **real or fake** using Machine Learning techniques.
It analyzes the text of news content and predicts its authenticity.

---

## 🚀 Features

* Detects fake vs real news from text input
* Built using Python and Machine Learning
* Simple web interface using Streamlit
* Uses dataset-based training

---

## 🛠️ Tech Stack

* Python
* Scikit-learn
* Pandas
* NumPy
* Streamlit

---

## 📂 Project Setup Instructions

### 1️⃣ Clone the repository

```bash
git clone https://github.com/your-username/fake-news-detection.git
cd fake-news-detection
```

---

### 2️⃣ Download the Dataset

Download the dataset manually from a trusted source (e.g., Kaggle).

After downloading:

👉 Move the dataset files into the project folder
👉 Place them inside the **data/** directory

Example:

```
fake-news-detection/
│
├── data/
│   ├── Fake.csv
│   └── True.csv
```

---

### 3️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

---

### 4️⃣ Run the Streamlit App

```bash
streamlit run app.py
```

---

## 💻 How to Use

1. Run the Streamlit app
2. Enter news text into the input box
3. Click **Predict**
4. The model will classify the news as Real or Fake
